# Alert

Section Theme blog provides an [inbuilt Alert component](https://mantine.dev/core/alert/) with mantine UI.

## Example

```javascript
<Alert>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras facilisis quis tellus eget congue. Curabitur ullamcorper faucibus efficitur.
</Alert>
```

## Color and Background

You can change Background and font color in Alert component.

```javascript
<Alert title="Lorem ipsum" color="red" variant="outline">
   Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras facilisis quis tellus eget congue. Curabitur ullamcorper faucibus efficitur.
</Alert>
```

## Icon
You can change svg Icon in Alert component.

```javascript
<Alert icon={<IconAlertCircle size="1rem" />}>
   Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras facilisis quis tellus eget congue. Curabitur ullamcorper faucibus efficitur.
</Alert>
```

## Radius
You can change Radius in Alert component.

```javascript
<Alert title="Lorem ipsum"  radius="md" variant="outline">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras facilisis quis tellus eget congue. Curabitur ullamcorper faucibus efficitur.
</Alert>
```
