# Iframe

The section theme comes with inbuilt Iframe support. Iframe component based on [react-iframe](https://www.npmjs.com/package/react-iframe) library.

## Example
```javascript
<Iframe width="560" height="315" src="https://www.youtube.com/embed/twk3eLJtZuY" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen />
```
