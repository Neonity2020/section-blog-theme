# Space 

Add horizontal or vertical white space in MDX file. The Space component built in top of [Mantine space component](https://mantine.dev/core/space).
## Example

```jsx
<Space />
```
## Height

Adding height in `Space` component.

```jsx

// h = 'xs' | 'sm' | 'md' | 'lg' | 'xl' | string | number;

<Space h="md"/>

```
