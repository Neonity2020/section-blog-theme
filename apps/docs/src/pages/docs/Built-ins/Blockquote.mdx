# Blockquote

You can add Blockquote component inside your MDX file. The Blockquote component is built based mantine UI [Blockquote component](https://mantine.dev/core/blockquote/).

## Example

```javascript
 <Blockquote cite="– Forrest Gump">
      Life is like an npm install – you never know what you are going to get.
</Blockquote>
```
## Color
You can change the color in blockquote component.

```javascript
<Blockquote color="pink" cite="– Forrest Gump">
   Life is like an npm install – you never know what you are going to get.
</Blockquote>
```
## Icon

You change the default icon in Blockquote component.

```javascript
 <Blockquote  icon={<IconFlame size="1.5rem" />} cite="– Forrest Gump">
   Life is like an npm install – you never know what you are going to get.
</Blockquote>
```
