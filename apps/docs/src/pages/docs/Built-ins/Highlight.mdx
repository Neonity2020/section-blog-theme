# Highlight

The highlight component helps to highlight the given string with the HTML mark tag in MDX. [Highlight component](https://mantine.dev/core/highlight/) built top on mantine UI.
## Example
```javascript
<Highlight highlight="this">
      Highlight This, definitely THIS and also this!
</Highlight>
```

## Color

```javascript
<Highlight highlightColor="red" highlight="this">
    Highlight This, definitely THIS and also this!
</Highlight>
```

## Multiple Highlight

```javascript
<Highlight highlight={['this', 'that']}>Highlight this and also that</Highlight>
```
