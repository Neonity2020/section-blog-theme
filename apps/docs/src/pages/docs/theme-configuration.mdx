## homePageAsAuthor
You can convert the home page into your Author page. It is helpful when somebody creates their own portfolio blog. If you have the home and author pages both different, then you can use `homePageAsAuthor=false`. You can also add a custom URL for the author. 

OptionOption       | TYPE          | Require|  Description 
------------- | ------------- | ------- | -------------
homePageAsAuthor  | `string<URL> | boolean` |  `true` | Convert the home page into author page and default vale is `true`

## dateFormat
Change the date dateFormat in your blog site. By default date type is `MMM DD, YYYY`, and it built the use of the day.js library. Check out all the [date format lists](https://day.js.org/docs/en/display/format).

 Option       | TYPE          | Require|  Description 
------------- | ------------- | ------- | -------------
dateFormat  | `string`  |  `false` | Change the date fromat 

## siteURL
Site URL is help in SEO. It takes your production URL by default it uses the local development `http://localhost:3000` URL

Option       | TYPE          | Require |  Description 
------------- | ------------- | ------- | ------
siteURL | `string`  | `true` | Use for SEO


## defaultSeo
defaultSeo is part of the [next-SEO]((https://github.com/garmeeh/next-seo)) package. You can add a default Seo Meta tag for your blog.

Option       | TYPE          | Require |  Description 
------------- | ------------- | ------- | ------
defaultSeo | [`DefaultSeo`](https://github.com/garmeeh/next-seo#default-seo-configuration)  | `true` | Config SEO Meta Tag


## logo
You can change the logo on your blog. You can change and edit text and SVG.

Option       | TYPE          | Require |  Description
------------- | ------------- | ------ | -------------
logo.text  | `string`  | `false` | Name of logo
logo.svg   |  `reactNode` | `false` | svg logo


##  primary_navigation
Primary Navigation is used on your header; you can edit and change it. 


Option       | TYPE          | Require |  Description
------------- | ------------- | ------ | -------------
 route | `string`  | `true` | path or url 
 name | `string`   |  `true` | name of url


##  secondary_navigation
Primary Navigation is used on your footer; you can edit and change it. 

Option       | TYPE          | Require |  Description
------------- | ------------- | ------ | -------------
 route | `string`  | `true` | path or url 
 name | `string`   |  `true` | name of url


## social_links
Social media icon show on your website header.

Option       | TYPE          | Require |  Description
------------- | ------------- | ------ | -------------
 url | `string`  | `true` | full link of social media  
 name | `Facebook Twitter Youtube Linkedin Github Instagram` |  `false` | name of url 
svg |  `reactNode` | `false` | social media icon
