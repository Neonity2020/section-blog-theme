The section theme inbuilt support multiple layouts for the blog. You can handle it with front matter. 

## Front matter 
Section theme supports the common front matter for your blog that helps you with SEO.

```markdown  filename="front matter example"
---
type: page
title : my title
expect: my description
image:
  - https://cdn.pixabay.com/photo/2016/05/31/10/52/contact-us-1426589_1280.png
  - https://cdn.pixabay.com/photo/2014/08/05/10/27/iphone-410311_1280.jpg
---
```

## Home Page
You can set up a blog home page with the Section Theme. It is unique For your site. 

![Home Page](/Home.png "Home Page")


```markdown {2} filename="index.mdx"
---
type: home
title: Home Page - Rajdeep Singh
except : Home page 
image:
  - url : https://cdn.pixabay.com/photo/2023/05/21/19/45/rose-8009275_1280.jpg
    alt : Pixabay image
---
```

## Basic Page
![Page](/Page.png "Page")

You can create basic or regular page like about, contact, etc.

## Authors Lists Page

You can show all the authors on a single page as an author's list. 

![Authors Page](/Authors.png "Authors Page")

```markdown copy {2} filename="index.mdx"
---
type: authors
title: Authors List
except: Sit culpa elit excepteur ea Lorem ullamco eiusmod ex ut ea incididunt minim. Cillum eiusmod fugiat cupidatat.
---

```

## Author Page
Create an author page for a single author, where you show author about, social links and related article to author.

![Author Page](/Author.png "Author Page")

```markdown {2} filename="author-name.mdx"
---
type: author
name: Annie Dean
except: Sit culpa elit excepteur ea Lorem ullamco eiusmod ex ut ea incididunt minim. Cillum eiusmod fugiat cupidatat.
date: 2022-01-09T09:01:55.077Z
job : JavaScript 
tags: 
   - Annie Dean
image: https://cdn.pixabay.com/photo/2023/04/30/09/43/flower-7960192_1280.jpg
social:
  - name: github
    url: https://github.com/frontendweb3
  - name : twitter
    url: https://twitter.com/Official_R_deep 
---
```


## Posts Page
Show all the posts or articles in current folder.

![Posts Page](/Posts.png "Posts Page")

```markdown {2} filename="index.mdx"
---
type: posts
title: Posts List
expect: Our blog posts
---
```
